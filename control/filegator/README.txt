
FileGator v4.2.4

Documentation is located under documentation folder

Quickstart:

- Adjust configuration options inside configuration.php file
- Administrator account: admin/admin123
- filegator/config/config.json and filegator/repository/ must be writable
- If you want to store users inside database use database.sql to create initial table and switch use_database to `true` (configuration.php)

Compatibility:

- PHP Versions 5.2 or higher + Apache web server. Consult your server administrator or hosting provider about this.
- Browsers: Microsoft IE7, IE8, IE9, IE10, Firefox, Chrome, Safari.
