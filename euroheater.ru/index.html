Основные направления

  • Производство аналогов теплообменников оригинальных моделей
    Различные аналоги теплообменников с медными и стальными коллекторами для приточно-вытяжных установок на производственной площадке

    Мы хорошо знаем техническую документацию многих европейских и российских производителей вентиляционных установок. У нас есть опыт изготовления изделий в соответствии с ней.

  • Нестандартные теплообменники на заказ по вашим размерам, характеристикам
    Фреоновый испаритель из нержавеющей стали. Морское исполнение.

    При необходимости мы можем изготовить и то, что не имеет аналогов ни в Европе, ни в России. Все вопросы обсуждаются, нужно только ТЗ.

  • Калориферы (нагреватели), охладители, теплообменники для вентиляции
    Крупный теплообменник-охладитель 14 рядов, для приточно-вытяжной установки

    Уникальная вещь: на заказ по специфическим данным ТЗ. Со смещением впускного патрубка впускного коллектора. Для монтажа в существующую вент установку 2.60м на 2.20м. Возможно изготовление до 22 рядов с диаметром трубки 16мм толщиной стенки 1мм.

  • Производство теплообменников для пищевой, нефте-химической, металлургической, медицинской, атомной и других отраслей промышленности
    Теплообменники из нержавеющей стали AISI 316

    Теплообменники из нержавеющей стали AISI 316, предназначенный для охлаждения азотоводородной смеси с парами кислоты 740-800 град С, в агрегате электроизоляционного покрытия линии холодного проката металлургического комбината.

Будем полезны вам:

  • Вы эксплуатируете оборудование теплообмена, случилась авария и нужна замена.

    Из опыта: вышел из строя калорифер вентиляции в офисном центре - сделаем аналогичный по размерам и характеристикам.

  • Вы сервисная, строительная, производственная компания
    и заказываете теплообменное оборудования
    для своего проекта.

    Из опыта: Фирма занимается обслуживанием установки по сушке крахмала на кондитерской фабрике, заказывет теплообменник по индивидуальным размерам, материалам и характеристикам для требуемых условий производства.

  • Участвуем в тендерах для госкомпаний по ФЗ №44 и ФЗ №223

  • Участвуем в коммерческих тендерах.
